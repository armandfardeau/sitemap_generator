# Decidim::SitemapGenerator

A sitemap generator for Decidim.

## Usage

SitemapGenerator will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-sitemap_generator"
```

And then execute:

```bash
bundle
```
